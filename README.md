# Installation:
```sh
1. Install NodeJS 12.x
2. npm install moment-duration-format
3. Use your bot token generated at Discord Devoloper website
4. Paste the bot token on line 2 in the \database\configs.js file
5. Copy the ADMIN ID where the attack command is written and change the ADMIN at line 12 of the file \database\configs.js
6. Copy the Room ID where the attack command is written and change the Room at line 14 and 15 of the file \database\configs.js
7. Run the command to start the bot by typing node index.js
8. Let's have fun
